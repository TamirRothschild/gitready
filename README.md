## How To Use 
###### Terminal:

git clone https://github.com/TamirRothschild/gitready

cd 'gitready/dev/preview' [WHILE IN DEV]          

./gitready2p

## How To Build
###### Terminal:
git clone https://github.com/TamirRothschild/gitready

cd 'gitready/dev/preview' [WHILE IN DEV]

gcc -o gitready2m 2.c

./gitready2m
